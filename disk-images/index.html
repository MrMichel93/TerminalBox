CTF Challenge Disk Images

Download disk images for CTF challenges. Each disk image is a compressed archive containing a complete challenge environment ready to run in the TerminalBox Linux emulator.

About These Disk Images

Each disk image is a .tar.gz archive containing:

  • Challenge setup script (setup.sh)
  • Challenge documentation (README.md)
  • All necessary files and configurations

Total size: ~28KB for all 8 challenges (extremely lightweight)
Format: tar.gz (gzipped tar archive)
Compatible with: Linux, macOS, Windows (with appropriate tools)

How to Use

Download and Extract

# Download a challenge (or use the download buttons above)
wget https://yourusername.github.io/TerminalBox/disk-images/challenge1-hidden-files.tar.gz

# Extract the challenge
tar -xzf challenge1-hidden-files.tar.gz

# Navigate to the challenge directory
cd challenge1-hidden-files

# Read the challenge instructions
cat README.md

# Run the setup script
bash setup.sh
